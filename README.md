# Elements — Periodic Table

一个精美的元素周期表网页应用，采用液态玻璃界面设计。

## 功能特性

- 📊 **完整的元素周期表** - 包含118种元素
- 🎨 **美观的液态玻璃界面** - 采用现代玻璃态设计风格
- 🔍 **元素搜索功能** - 快速查找元素
- 🏷️ **分类筛选** - 按元素类别筛选查看
- 📱 **响应式设计** - 适配各种屏幕尺寸
- 🎯 **交互式模态框** - 点击元素查看详细信息
- 🔬 **3D原子模型** - 使用Three.js渲染的原子结构可视化
- 💡 **悬停提示** - 鼠标悬停显示元素名称和分类

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计，包含玻璃态效果
- **JavaScript** - 交互逻辑
- **Three.js** - 3D原子模型渲染
- **Google Fonts** - 字体支持

## 文件结构

```
element/
├── index.html      # 主页面结构
├── style.css       # 样式文件

```

## 使用方法

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html` 文件
3. 开始探索元素周期表！

## 功能说明

### 搜索
使用顶部搜索框按元素名称、符号或原子序数搜索。

### 分类筛选
点击顶部图例中的类别可以筛选显示该类别的元素。

### 查看详情
点击任意元素卡片打开详情模态框，查看：
- 原子质量
- 发现年份
- 电子排布
- 常温状态
- 原子序数和分类
- 3D原子模型（可拖拽旋转）

## 元素分类

- 碱金属 (Alkali Metals)
- 碱土金属 (Alkaline Earth Metals)
- 镧系元素 (Lanthanides)
- 锕系元素 (Actinides)
- 过渡金属 (Transition Metals)
- 后过渡金属 (Post-transition Metals)
- 类金属 (Metalloids)
- 非金属 (Nonmetals)
- 卤素 (Halogens)
- 稀有气体 (Noble Gases)
- 未知性质 (Unknown Properties)
